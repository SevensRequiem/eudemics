# Defense

Military structure, vetting, and integration with the node system.

## Military Node Offices

Military is not a separate government. It is an office within each node, integrated with civilian governance. Nodes are uniform in structure but have specialized offices: governance, military, engineering, medical, etc.

An engineer and a strategist sit in the same node but different offices. The engineer has technical insight the strategist lacks. The strategist has operational awareness the engineer lacks. Both contribute to the node's function. Neither dominates.

## Resource-Appropriate Assignment

Military resources are matched to threats, not uniformly distributed:
- Nation-state cyber attack: nodes with cybersecurity resources handle it, not agricultural nodes
- Local conflict: neighboring nodes contribute, distant nodes only if needed
- Ballistic/asteroid/nuclear threat: nodes with relevant defense infrastructure activate
- Regional conflict: more nodes engaged, but each node decides if contributing would detriment its own efficiency

The system appropriately routes threats to the nodes best equipped to handle them.

## Vetting

All military personnel are extremely vetted. Competency requirements are higher than civilian governance for sensitive roles. Access to weapons systems, intelligence, and strategic planning requires demonstrated capability AND demonstrated reliability.

## Treason

Genuine traitors, with evidence-supported detriment to the whole or its security, face severe punishment. This extends to civilians in sensitive assigned positions.

The evidence standard is high: documented harm, not disagreement or incompetence. Whistleblowing is explicitly NOT treason (see [[justice/restoration]]).

AI-based traitor assessment requires human VRF committee final determination. AI can flag patterns, humans decide. See [[decision-making/accountability]] for AI coup prevention.

## Node-Level Defense

Nodes in dangerous areas (border regions, contested zones, disaster-prone) have proportionally larger military offices. Cities prepare defenses against ballistic, asteroid, and nuclear threats. Coastal nodes prepare for naval and climate threats.

Each node's defense posture is tailored to its actual risk environment, not a one-size-fits-all model.
