# Statutory Law

Specific operational rules enacted under constitutional authority.

## Key Statutory Areas

- **Substance regulation** - tier classifications, waiver requirements, credit system rules, public intoxication prohibition. See [[autonomy/liberty]].
- **Genetic correction mandates** - which corrections are mandatory, opt-out consequences, reproductive tracking requirements. See [[technology/ethics]].
- **Advertisement regulation** - limits on invasive advertising, prohibition of manipulative psychological techniques, mandatory media categorization. See [[knowledge/information-flow]].
- **Corporate power limits** - lobbying is illegal, corporate reach regulated, no corporate capture of governance. See [[knowledge/information-flow]].
- **Land ownership caps** - dynamic limits based on population density and infrastructure needs. See [[resource-systems/ownership]].
- **Birth tracking** - untracked births are mildly illegal. See [[technology/ethics]].
- **Data sovereignty** - legal framework for data packs, state tracking scope, override key authorization. See [[autonomy/privacy]].
- **IP and copyright** - TTL on copyright, MIT-license approach, patent prohibition, inventor recognition. See [[resource-systems/distribution]].
