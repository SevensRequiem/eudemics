# Migration

Movement of people between nodes and into the system.

## Free Movement

Free movement within large node areas (continents). Cross-continent movement requires updating your ID and data pack so data stays relevant to your new node.

## Node Desirability

"One node is more desirable" is a system failure, not a migration problem. The whole network is optimized for desirability everywhere. If a node underperforms, resource allocation and infrastructure investment flow toward it. Population concentration that breaks the federated model is prevented by making all nodes worth living in.

## External Immigration

People from non-eudemics nations must be educated on the system before integration. Not a citizenship test, but genuine understanding of how the model works, what their rights and obligations are. The goal is informed participation, not gatekeeping.

## Asylum

Asylum seekers use the same whistleblower protection mechanism: choose any node, that node is legally obligated to protect you. See [[justice/restoration]].
