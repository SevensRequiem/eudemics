# Eudemics

This is a conceptual model - a thought experiment in governance optimization oriented toward the human condition. There is no codebase, no software to build, test, or run.

## Nature of the Project

- **Model, not software.** This repo contains ideas, not code. No build commands, no tests, no deployable artifacts.
- **No versioning milestones.** There is no "v1" or release target. The model evolves through continuous refinement - ideas get added, challenged, revised, or discarded as understanding deepens.
- **Foundation.** Draws from technocracy's resource/energy accounting as a starting point, then diverges toward optimizing for human wellbeing rather than any particular group or ideology.

## Working With This Repo

When contributing to this project, treat changes as iterative refinements to a living document. No change is "done" - everything is subject to revision as the model improves.
