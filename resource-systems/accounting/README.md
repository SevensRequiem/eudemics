# Accounting

How resources are measured, priced, and allocated.

## Token Economy

The economy runs on digital tokens and physical coins issued by the node network, not a central bank. Distribution is across all nodes.

There is no gold backing. The backing is the resource allocation system itself. Tokens represent purchasing power within a system that manages real resources. The "backing" is that the system guarantees the token buys real things: food, housing, vehicles, services.

Physical coins exist for resilience. If digital systems fail (EMP, disaster, infrastructure loss), the economy does not stop entirely. See [[security/resilience]].

## Resource Pricing

Prices are informed by:
- Energy cost of production/extraction
- Environmental cost at point of impact (not just extraction)
- Current supply relative to demand
- Renewal rate for renewable resources
- Local environmental carrying capacity

This is evidence-based pricing, not market speculation. Prices reflect reality, not sentiment. See [[ecology/sustainability]] for how environmental capacity constrains pricing.

## Dynamic Resource Limits

Resource availability is a live function, not static caps:

Available resource footprint = f(known supply, extraction rate, environmental carrying capacity, renewal rate, population)

- **Never below baseline** - even in scarcity, everyone gets enough. The system must predict shortfalls and pre-adjust: build reserves, throttle extraction early.
- **Discovery events** - a new lithium vein or asteroid mining haul dynamically increases limits and reduces prices for those resources.
- **Environmental veto** - even if supply is unlimited, environmental capacity constrains. Asteroid mining floods the market with raw materials, but ocean/shoreline carrying capacity limits yacht production regardless of material availability.
- **Local environment weighting** - a region with fragile ecosystems has tighter limits than a robust one. Resource pricing reflects environmental cost at point of impact.

## State Funding

The state is funded through allocation, not taxation. The network manages resources directly. State operations (infrastructure, education, health, military, research) are budgeted through evidence-based and predictive allocation, per node and global.

A node can deny sending resources to another. But if the requesting node and other nodes vote that help is needed, the denying node is overridden. The system serves the whole. See [[decision-making/consensus]].
